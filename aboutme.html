Scratch vs. Python





It was fun learning about the similarities between Scratch and Python. 
Scratch used a more visual way to display the code, while python seems more mature and used in the real world. Both use things
like variables and help a program run. Python can vary stylistically in the way it is written, while Scratch does not.
